## 任务状态说明

| 状态图标 | 状态名称 | 说明 |
|----------|----------|------|
| ⏳ | 待开始 | 任务已规划，等待开始 |
| 🎨 | 进行中 | 任务正在执行 |
| ⏸️ | 已暂停 | 任务暂停，等待后续继续 |
| ⚠️ | 有风险 | 任务执行中遇到风险 |
| ❌ | 已延期 | 任务未按时完成 |
| ✅ | 已完成 | 任务已完成并验收 |

## 优先级说明

| 优先级 | 说明 |
|--------|------|
| P0 | 关键路径任务，必须按时完成 |
| P1 | 重要任务，尽量按时完成 |
| P2 | 一般任务，可灵活调整 |
